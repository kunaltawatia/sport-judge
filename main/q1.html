PROBLEM A: Black Obstacles

Rohan is obsessed with white.He buys a chess board of n x n dimensions and it is all white,his younger brother wanted to correct the chess board so he randomly started painting some of the white cells into black.

When Rohan came to know all this he started crying and the only way to quiet him is giving the largest possible white square board by cutting it from the already spoiled white chess board.

Let the acquirable white chess board be of dimension m x m.Calculate the value of m.

Input Format :

First line contains a single integer n.

The next n lines have n space-separated boolean values 0 for white and 1 for black.

This shows the board after Rohan's younger brother spoils the board.

Constraints:
For 20%of score
n≤10

For 80%of score
n≤1000


Output Format:

Print size of largest white piece left
