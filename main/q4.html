PROBLEM D: ScoreKeeping

Two teams are playing Basketball organised by WeBug Community of IIT Jodhpur. On every score, team is rewarded with some rating and not points. As we have seen that self-baskets i.e basket on one's own side do happen so for these conditions rating can also have a negative value.

Given an array of ratings of n scores, A captain's final task is to calculate the maximum subarray sum,i.e., the largest possible sum of a sequence of consecutive values in the array.

Both the captains then report their maximum subarray sum and the team having greater sum wins.

If the 1st team wins print 1 else print 2

For simplicity both the teams have n number of baskets and we have a clear winner i.e the game is not draw.

Input Format :

First line contains value of n.

Second line have n space-seperated integer values of ratings of each basket by 1st team.

Third line have n space-seperated integer values of ratings of each basket by 2nd team.

Constraints:
For 20%of score
n≤10^3

For 80%of score
n≤10^5


Output Format:

Print a single integer, Either 1 or 2
